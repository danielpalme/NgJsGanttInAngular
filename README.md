# Angular application hosting AngularJS gantt component

Demo Angular 21 application which hosts an [angular-gantt](https://github.com/angular-gantt/angular-gantt) component that is based on AngularJS using the [UpgradeModule](https://angular.io/api/upgrade/static/UpgradeModule).
